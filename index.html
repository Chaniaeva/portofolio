My Portofolio

Room Booking System

The room booking system is a web-based system that is still being used at OJK regional offices 3, Central Java and the Special Region of Yogyakarta. This system is used to carry out the process of borrowing room in the office. Previously, the process was done manually by taking notes on the book provided. This system is present to help the administration division track and note the history of room bookings, making the room booking process computerized, more effective, and efficient.

Grade Conversion

This application is a simple website-based GPA value conversion application. Required data is the name of the course, semester credit unit, and grade in the form of 0-100 for each course. The output of this application is a grade point average in the form of 0-100, 0-4, and letters.

Cashier

This application is a simple website-based cashier application. Required data is the name of the goods, quantity, price, and percentage discount. The output of this application is the amount that must be paid in the process of purchasing the goods.
